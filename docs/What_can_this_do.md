# 它能做什么？

以下内容展示如何将本项目配合 RSSHub、RSSBot 两个项目，实现爬取网络内容并分类推送。

1. 部署 RSSHub 和 RSSBot，请参见这两个项目的文档。
   对于 RSSBot，可以尝试使用本项目维护者提供的[分支版本](https://github.com/finall1008/rssbot)，相对更适用于配合 RSSHub 使用。
2. 参考部署说明部署本项目。
3. 新建一个频道（“中继频道”），添加 RSSBot 和本项目的 Bot 为管理员，在 RSSBot 中订阅 RSSHub 提供的爬取内容。
4. 新建一个群组（“监视器群组”），添加本 Bot 为管理员。
5. 设置从中继频道到监视器群组的自动转发规则。
6. 可以开始使用了，对于监视器群组中自动转发的来自 RSSBot 的每一条消息，您都可以对它们进行分类、推送到指定目标的操作。
